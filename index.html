Stay Unlocked

Keep this website open in a separate window to prevent your machine from going to sleep. This can primarily be used for work environments where you want to be able to monitor something on a machine that you can't change the sleep settings on, without having to remember to keep moving the mouse so you don't have to login again. This is not intended to be used as a device that prevents an actual "idle" state, it only prevents sleep - DO YOUR WORK!

Requirements

  • This page must be in a window by itself, no other tabs
  • This does not currently work if you're running a laptop on battery
  • This does not seem to work in Safari
