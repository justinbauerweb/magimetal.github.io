Stay Unlocked

This website helps a user prevent their machine's display from going to sleep. It does not generate "activity", but rather acts like you're playing a fullscreen video (even though you're not).

Users can continue to work on multiple machines at once without missing important emails or messages from other Communication Software (Microsoft Teams, Slack, Discord, Mattermost, etc).

You must keep this website open in a separate window for the functionality to work, it can't be in a tab in the background or minimized.
