Stay Unlocked

Keep this website open in a separate window to prevent your machine from going to sleep.


This can primarily be used for work environments where you want to be able to monitor something on a machine (like Teams Chat or Slack), but you can't change the sleep settings on the machine. There are probably a bunch of other viable uses as well, but this was my primary use-case.


This is not intended to be used to prevent an actual "idle" state, or fake use activity - it simply prevents your machine from going to sleep.

Requirements

  • This page must be in a window by itself, no other tabs
  • This does not currently work if you're running a laptop on battery
  • This does not seem to work in Safari
